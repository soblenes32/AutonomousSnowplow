*** About ***

This is a Spring 4 Boot project quickstart template. This version contains integrations of the following technologies:
-> Gradle-based build
-> Java-based Spring configuration
-> Webjars configuration of javascript dependencies
-> Angular 1.x libraries and starter code in a single-page webapp configuration
-> Spring security ldap-based configuration
-> JNDI-based MySql integration
-> Hibernate-based JPA sample
-> Spring Data Rest project integration

*** Other notes ***

If using email...
To prevent java from directing smtp connections via IPv6, add the following to the execution line for tomcat: -Djava.net.preferIPv4Stack=true
